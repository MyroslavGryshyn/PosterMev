# PosterMev

How to run app:

1) Clone repository: 

    git clone --single-branch https://github.com/MyroslavGryshyn/PosterMev.git

2) Install requirements: 

    pip install -r requirements.txt

3) Migrate with your db:

    python manage.py migrate

4) Run app:

    python manage.py runserver

5) Enjoy =)
